這個專案是要建立一個 Chrome Extension，要能夠錄製使用者在網站上操作的步驟，並使用 LM Studio server 的 llm 解析並產生 Playwright 操作的步驟，目標如下：

1. 按下去之後，會出現畫面，顯示開始錄製的按鈕，分為一次錄製模式以及逐步錄製模式
2. 按下去之後進到錄製模式，網頁上會出現一組可拖移的 toolbar 顯示暫停錄製或結束錄製
3. 逐步錄製模式，紀錄使用者按下的按鈕，即時傳送到llm 處理，產生相對應的 playwright script
4. 一次錄製模式，則記錄下使用者按過的所有按鈕，完畢後在背景逐步送給 llm 處理產生 playwright script，以應付有即時性的網頁
5. 產生的 script 可以選擇 js 或是 python 格式
