Usage Notes

LM Studio (local, OpenAI compatible): Local AI server, default URL http://localhost:1234/v1/chat/completions. No API key required. Pick from the dropdown of models loaded in LM Studio.
Want to use Ollama or another local endpoint? Keep the LM Studio provider and change the API URL to http://localhost:11434/v1/chat/completions (the protocol is 1:1 compatible).
OpenRouter (cloud aggregator): One API key routes to Claude / GPT / Gemini / Llama / DeepSeek and more. Default URL https://openrouter.ai/api/v1/chat/completions. Requires an OpenRouter API Key. The model name field is a free-form text input with autocomplete suggestions; type any model id supported by OpenRouter, e.g. anthropic/claude-sonnet-4, openai/gpt-5, google/gemini-2.5-flash.
